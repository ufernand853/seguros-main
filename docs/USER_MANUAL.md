# Manual de usuario – Portal Gestión de Seguros

Este manual resume cómo iniciar sesión, navegar por el dashboard y utilizar las principales secciones del portal.

## 1. Acceso al portal
1. Abre el portal e ingresa tu **Correo corporativo** y **Contraseña**.
2. Haz clic en **Iniciar sesión**.
3. Verás el dashboard principal si las credenciales son correctas.

## 2. Dashboard: mosaicos y funciones
Cada mosaico abre un módulo distinto:
- **Pipeline de pólizas**: flujo de pólizas en curso (nuevas, en emisión, emitidas, renovaciones) para priorizar y detectar cuellos de botella.
- **Producción & comisiones**: primas emitidas y comisiones por periodo/aseguradora/productor; útil para control de ingresos y liquidaciones.
- **Agenda de renovaciones**: calendario/listado de pólizas próximas a vencer para anticipar renovaciones y evitar caídas de vigencia.
- **Seguimiento de gestiones**: tablero de tareas y gestiones pendientes (llamadas, emails, requerimientos) con responsables y avances.
- **Clientes**: alta, búsqueda y edición de clientes; punto de partida para la ficha integral o para iniciar pólizas y siniestros asociados.
- **Aseguradoras**: catálogo de compañías y ramos; administra acuerdos, métricas (activas, primas, siniestralidad) y ramos disponibles.
- **Ficha integral cliente**: vista 360° del cliente con datos, pólizas vigentes, siniestros históricos, documentos y gestiones.
- **Registro de siniestro**: registro guiado de denuncias; valida póliza/vigencia, captura datos de contacto y prepara notificaciones internas/externas.
- **Configuración**: ajustes generales (catálogos, usuarios/roles, parámetros de negocio, canales y plantillas de notificación).
- **Logout**: cierra tu sesión de forma segura.

## 3. Flujos principales

### 3.1 Clientes
1. Ingresar a **Clientes**.
2. Buscar por **Nombre, RUT/Documento o ciudad**.
3. Acciones rápidas: **+ Nuevo cliente**, **Ver ficha integral**, **Alta póliza & siniestro (base real)**.
4. Si no hay resultados, usa **Crear “Nuevo cliente”** para agregar un registro.

### 3.2 Aseguradoras
1. Abrir **Aseguradoras**.
2. En **Catálogo de compañías**, revisa métricas (compañías activas, primas anuales, pólizas vigentes, siniestralidad promedio) y usa **+ Nueva aseguradora** para crear una.
3. En **Catálogo de ramos**, consulta ramos disponibles y agrega nuevos con **Nuevo ramo** + **Agregar ramo**.

### 3.3 Registro de siniestro
1. Entrar a **Registro de siniestro**.
2. Completar **Datos del asegurado**: Cliente, Documento, Asegurado, Email de contacto, Teléfono y Responsable interno.
3. Verificar el **Resumen del reporte** (Cliente, Documento, Póliza, Aseguradora, Vigencia, Prioridad, Canal y Notificaciones).
4. Guardar/registrar según los controles disponibles en la instancia.

## 4. Consejos de uso
- Navega siempre desde el dashboard para acceder rápido a cada módulo.
- Antes de registrar un siniestro, confirma que el cliente y sus pólizas estén cargados para vincularlos sin retrabajo.
- Mantén actualizado el catálogo de aseguradoras y ramos para que los formularios y métricas reflejen la operación real.
- Usa **Logout** al finalizar para proteger tu sesión.
